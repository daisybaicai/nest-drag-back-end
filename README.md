
# Nest-drag-back-end

可视化平台的后端

## Description

基于Nest.js的后端。数据库使用MySQL.

## Installation

```bash
$ npm install
```

## Running the app

首次使用需先下载对应的数据库配置文件（test4.sql），以及在config/db.ts配置相应的数据库。

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
